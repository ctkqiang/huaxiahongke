## 主要功能

该应用集成了以下工具模块：

- 分布式拒绝服务攻击 (DDoS)
- 蓝牙工具
- 无线网络 (Wi-Fi) 工具
- 监控摄像头检测
- 消息加密
- 终端控制台
- 通信工具
- 射频识别 (RFID) 模块
- IMSI 捕捉器
